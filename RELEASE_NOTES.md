# WiFiShield v1.0 - Initial Release

## What's Included

- Complete ESP32 firmware source code
- Evil Twin attack implementation
- Credential capture system
- Captive portal simulation
- Aircrack integration
- Web admin interface
- Bilingual documentation (English + Indonesian)

## Quick Start

\\\ash
git clone https://github.com/Harriiee/wifishield.git
cd wifishield
python -m platformio run -e esp32
python -m platformio run -e esp32 --target upload
\\\

Connect to WiFi \WiFiShield\ (password: \12345678\) and access \http://192.168.4.1/admin\

## Requirements

- ESP32 Development Board
- USB Cable
- Python 3.8+
- PlatformIO
